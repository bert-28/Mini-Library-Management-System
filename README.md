## Design Decisions
* **Layout:** Used a split-pane design for forms and tables to reduce page scrolling.
* **UX:** Implemented modal pop-ups for 'Edit' functions to keep the user on the current page.
* **Color Palette:** Used a neutral professional palette with high-contrast buttons for primary actions (like "Borrow Book").

## How to Run
1. Clone the repo.
2. Open `index.html` in your browser.
